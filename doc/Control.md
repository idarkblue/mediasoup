# 开始创建房间

## worker.createRouter
```json
    {
        "id":1,
        "method":"worker.createRouter",
        "internal":{
            "routerId":"routerId-0"
        }
    }
```

```json
    {
        "id":2,
        "method":"router.createAudioLevelObserver",
        "internal":{
            "routerId":"routerId-0",
            "rtpObserverId":"audioLevelObserver-0"

            "threshold":-80,
            "interval":800
```
